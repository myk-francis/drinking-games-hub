# Drinking Games Hub

Drinking Games Hub is an online platform where users can create, share, and play various drinking games. The application offers a wide range of games designed to be fun and engaging.

## Features

- **Game Selection**: Choose from multiple games including Truth or Dare, Higher Lower, Never Have I Ever, and more.
- **Player Management**: Easily add and remove players before starting a game.
- **Game Configuration**: Some games allow customization (e.g., number of rounds for Truth or Drink).
- **Room Creation**: Create private rooms to play with friends.
- **Shareable Links**: Share room links via QR code or direct URL to invite others.

## Getting Started

1. **Visit the App**: Head over to [Drinking Games Hub](https://drinking-games-hub.vercel.app/) to see the app in action.
2. **Explore Games**: Browse through the available games and select one that interests you.
3. **Create a Room**: Add players, configure settings if necessary, and create your room.
4. **Share the Game**: Generate a QR code or share the URL with friends to start playing together.

## Technologies

- **Framework**: Next.js
- **UI Components**: Tailwind CSS, Lucide React Icons
- **State Management**: React Query
- **Authentication**: TRPC for server-side operations
- **QR Code Generation**: qrcode.react
